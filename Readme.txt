This application is an interface for displaying the public user gists of GitHub users.
It is built using Angular 8, HTML5, CSS3, Bootstrap 5 and the GitHub gists API. 
When the app is run, a request is made to the API  and 100 gists are fetched and the results are rendered in the form of a table.
The user image, hyperlinked username, description and the github link to the gist has been displayed.
Pagination has been provided at the bottom of the table. Basic sort is implemented.
Ascending and descending icons have been provided the top of the table which on clicked sort the results on the basis of username.
